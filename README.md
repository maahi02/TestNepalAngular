# TestNepalAngular

- install node in your machine from  https://nodejs.org/en/
- after installation Node, 
- install  Angular CLI using Command Prompt
- use command:

- npm install -g @angular/cli   

- now open project directory in Command Prompt  ---- (e.g  D:\TestNepalAngular-master\NewTestAPP)
- (make sure your directory folder has angular.json file)

- now use command 
- npm install  (--- to install packages which is required for project)

- after installation 
- Use command to run project 
- ng serve -o    (---- ( to build project and open in browser))
